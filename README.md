# Arturs Developer Portfolio

Suggested repository name: `arturs-developer-portfolio`

This is a personal portfolio website built with React and Vite.
The project is designed as a presentation website for job applications, portfolio review, and client communication.

## What is implemented
- full-screen loading intro
- multilingual interface (English, German, Russian)
- sticky glass-style navigation
- developer introduction section with profile photo
- project showcase area with category tabs
- media slider with screenshots and video preview
- custom video controls and fullscreen mode
- contact section for portfolio presentation

## Tech stack
- React
- Vite
- JavaScript
- CSS

## Project purpose
This project was created as a personal portfolio website to present selected work, visual project demos, and a structured developer profile in a clean and modern format.

## Current content
- React project showcase with real media from the Pet Shop project
- prepared placeholders for Webflow and HTML/CSS/JS sections
- multilingual portfolio structure ready for future expansion

## How to run
```bash
npm install
npm run dev
```

## Build for production
```bash
npm run build
```

## Project structure
- `src/` — main application code
- `public/` — public assets, profile image, project media
- `dist/` — production build output

## Notes
This repository is prepared as a portfolio-ready project for GitHub presentation.
The website is intentionally structured to be easy to update with new screenshots, videos, and project categories over time.
